About GCDCollector

GCDCollector is a program to maintain a database of your personal comic collection. It is backed by a huge database of comics information that has been gathered over many years by the good folks at the Grand Comic Book Database Project (GCD) and made publicly availble.

This project is just getting underway, so there isn’t much here yet. But if you’re interested in helping out, give me a shout.

I was unsatisfied with the currently-available commercial solutions for managing a personal comic-book collection, so I set about writing my own software. That said, if you're looking for a solid (and supported!) comic database software—and especially given the very early state of this project—you should definitely try one or both of the following commercial offerings:

  • Collectorz.com (Comic)
  • ComicBase
